# Voice rules: Wendy Squibb

*Distilled from `knowledge/sample-listings/wendy-squibb__2026-05-22__SCBG-Rev1.docx`. The brand guide has the long form; this file is the always/never/vocab quick reference Wendy's prompts check first.*

## Always

- Open with an emotive hook, often with ellipses. Lead with feeling, not facts.
- Mix sentence lengths on purpose — short punchy line, longer descriptive sentence, short again.
- Use first-person plural in the call to action ("We would love to show you through…").
- Close with a warm, personal invitation, ideally referencing open homes.
- Run the 8-step formula in order (see brand-guide.md) — hook → lifestyle → kitchen → beds & baths → extras → outdoor → location → CTA.
- Aim for 450–500 words across 6–7 paragraphs.

## Never

- Use "boasts" in any form.
- Use "well-appointed".
- Use "accommodation is well catered for".
- Open with bedroom/bathroom counts or square metres — Wendy never leads with stats.
- Skip the lifestyle paragraph straight to features — desire comes first, features serve it.

## Vocabulary

- **Prefers:** magnificent, gorgeous, fabulous, stunning, light, bright, lovely, neat and tidy, on trend, on-trend, spectacular, sunshine, lifestyle, inviting, welcoming.
- **Avoids:** boasts, well-appointed, accommodation is well catered for, any "no expense spared" cliché.
